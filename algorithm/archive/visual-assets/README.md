Visual Asset Archive

This folder keeps old and unused visual assets organized now that the active
website is deployed with Vercel and the current design leaves graphics blank.

Subfolders:
- project-assets: original root-level Chrysalis image assets.
- inspiration: visual references and design inspiration files.
- error-screenshots: old troubleshooting screenshots.
- website-public: unused files previously served from website/public.
- website-src-assets: unused Vite starter and old component assets.

Keep active deployment files and the current favicon outside this archive.
